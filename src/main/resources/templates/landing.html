01 · Sobre a FatecRL

Tradição e inovação no ensino tecnológico

A FatecRL forma profissionais há mais de 35 anos. Com 7 cursos de tecnologia, mais de 4.200 alunos matriculados, somos referência em educação tecnológica na Baixada Santista.

O SIGA — Sistema Integrado de Gestão Acadêmica — foi desenvolvido internamente para modernizar e centralizar os processos de secretaria, eliminando papéis e reduzindo o tempo de atendimento em 70%.

02 · O sistema

O que o SIGA oferece

🗄️

Cadastro centralizado

Todos os dados dos alunos armazenados em banco PostgreSQL com segurança. ID único gerado automaticamente para cada registro.

🔍

Busca avançada

Encontre qualquer aluno por nome, CPF ou ID em segundos. Filtros inteligentes para agilizar o atendimento na secretaria.

🔒

Acesso seguro

Conexão criptografada com o banco de dados. Infraestrutura em nuvem com backup diário automático de todos os registros.

03 · Fluxo

Como funciona

Acesso à secretaria

O atendente acessa o sistema pelo navegador. Nenhuma instalação necessária — funciona em qualquer computador da instituição.

Preenchimento dos dados

Nome completo e CPF do aluno são informados no formulário. O sistema valida os dados antes de prosseguir.

Registro no banco

Os dados são salvos no PostgreSQL com um UUID único gerado automaticamente. O processo leva menos de um segundo.

Consulta imediata

O aluno já aparece na listagem e pode ser buscado por qualquer campo. Histórico sempre disponível para a equipe.
